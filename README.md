## 🛠️ Troubleshooting

### App does not open

- Check that you downloaded the latest release
- Make sure the file finished downloading
- If you got a `.zip`, extract it first
- Try right-clicking the app and selecting **Run as administrator**

### Windows blocks the file

- Right-click the file
- Select **Properties**
- If you see an **Unblock** option, turn it on
- Click **Apply**
- Open the file again

### BIOS files do not show up

- Make sure you selected the correct folder
- Check that the files are not still inside a zip archive
- Confirm the folder has read access
- Look for hidden files or nested folders

### Emulator still cannot find BIOS files

- Confirm the BIOS file name matches what the emulator expects
- Check the file type
- Make sure the BIOS file is in the correct BIOS directory
- Restart the emulator after changes

## 🧾 File Safety Tips

- Keep a backup of your BIOS folder
- Do not rename files unless you know the expected name
- Avoid mixing test files with your main BIOS set
- Use one folder structure across all your retro tools
- Store a copy on an external drive if your collection matters to you

## 🧪 Best Results

For the cleanest setup:

- Use the same BIOS folder for all compatible tools
- Keep system names consistent
- Remove duplicate files
- Sort files after each new download
- Check your folder before updating emulators

This keeps your setup easier to manage and reduces the chance of missing-file errors.
